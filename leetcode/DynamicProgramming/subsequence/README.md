## 总结
一般分为两种： 连续的子串以及非连续的子序列。
1. 设计dp方程，一般是根据字符的状态比较。例如回文，编辑距离问题 设计动态规划方程一般是针对字符串s[i] == s[j]和不相等两种情况。
2. 非连续时，需要更新不相等的dp状态。
